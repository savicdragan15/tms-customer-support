# TMS Customer Support Service

## Installation
* `cp .env.dist .env`
* `create database - tms_customer_support`
* `composer install`
* `php artisan migrate`
* `php artisan db:seed`
* `php artisan jwt:secret`

* `npm install`
* `npm run dev`

### Application serve
* `php artisan serve`
